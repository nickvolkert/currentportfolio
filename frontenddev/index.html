Cravath Website

Home page | Cravath

Careers page | Cravath

Cravath | Hubbard One

Project Role:

  • Owned front-end development after developer content was added, edited existing CSS around built out pages of site.
  • Styled and cross-browser reviewed site for bugs, fixed bugs as the came up.
  • Used CSS to style print version of site as well as PDF version of site.
  • Worked with Project Manager to remedy outstanding bugs per the client and Outside design firm's notes.
  • Project Launch: Edited and remedied site issues per browser until site launch.
  • Post Launch: Updated and remedied any bugs the client had noticed while using the site within the pages, the PDF and print versions of site.

Project Notes: A Few Challenges

  • "Add to Binder" Buttons that were extremely buggy browser to browser, had to use browser specific hacks to fix.
  • Pixel precise measurements with absolutley no shift between each supported browser (IE6, 7, 8, FF 3.6, Safari).
  • Getting the non-web-safe font "Arial Black"(used in all of the site headers) to work on pages by styling the CSS to "trick" the end users computer into displaying the font.
