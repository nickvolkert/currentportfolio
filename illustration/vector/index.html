Visual Attributes

The Visual Attribute project uses technical illustrations that appear in the search and browse flow of Grainger.com. The illustrations show different attributes of products for sale to help visually guide the user on their purchase. The project produced hundreds of illustrations, all created by me, and had a proven increase in revenue for the categories we created or refurbished line art for.
